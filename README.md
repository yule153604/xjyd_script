# 新疆移动任务挂机脚本

这是一个基于 Python (`asyncio` + `httpx`) 编写的自动化脚本，用于执行新疆移动相关的日常任务（签到、抽奖、浏览权益等）。

## ✨ 功能特性

本脚本整合了三个独立的任务模块，支持并发运行：

1.  **寻宝任务** (维语寻宝 中国移动APP & 微信公众号)
    *   每日自动签到
    *   支持多账号（APP 端 + 微信端）
    *   自动识别签到周期 (3/7/10/15 天)
2.  **每日抽奖** (新疆移动公众号)
    *   自动检测剩余抽奖次数
    *   自动执行抽奖
3.  **权益中心** (公众号权益中心)
    *   每日签到 + 首次抽奖
    *   自动完成浏览任务 (公众号/小程序)
    *   自动领取累计签到奖励 (7/14/21/28 天)

## 🛠️ 环境要求

*   Python 3.8+
*   依赖库：`httpx`, `pycryptodome`

安装依赖：
```bash
pip install httpx pycryptodome
```

## ⚙️ 配置说明

在使用前，请打开 `xjyd_tasks.py` 文件，在顶部的 **用户配置区域** 填入你的抓包数据。
> 提示：如果某个任务未配置 Cookie，请转到脚本底部的 `main` 函数（约 529 行），将对应的任务调用注释掉，避免报错。

### 1. 寻宝任务配置
*   **APP 渠道入口**：打开 "新疆移动" APP -> 首页切换 "维语版" -> 寻宝。
*   **微信渠道入口**：微信公众号 "maxrap10086" -> 右下角菜单 -> 寻宝。
*   **抓取说明**：抓取任意请求的 `Cookie`（参考 URL: `https://wap.xj.10086.cn/uyservice/server/xunBaoAct/xunBaoActivityNew/init`）。
*   **配置项**：
    *   在 `XUNBAO_ACCOUNTS` 列表中配置。
    *   **注意**：APP 渠道和微信渠道的 `User-Agent` 和 `Cookie` 是不同的，请分别抓取。
    *   支持多账号，直接复制列表中的字典块即可。

### 2. 抽奖任务配置 (maxrap10086公众号)
*   **入口**：关注 "maxrap10086" 微信公众号 -> 左侧菜单第一个 -> 进入活动页面。
*   **抓取说明**：抓取域名为 `wx.10086.cn` 的请求 `Cookie`（参考 URL: `https://wx.10086.cn/qwhdhub/lottery/remain`）。
*   **配置项**：`YDCJ_COOKIE`。

### 3. 权益中心配置
*   **入口**：
    *   方式一：关注 "新疆移动权益超市" 微信公众号 -> 中间菜单第二个 -> 签到领18元话费。
    *   方式二：浏览器直接打开链接 `https://wap.xj.10086.cn/quanyi/micro-page/everyDay.html`。
*   **抓取说明**：抓取域名为 `wap.xj.10086.cn` 的请求（参考 URL: `https://wap.xj.10086.cn/quanyi/earnBenefits/initPage`）。
*   **配置项**：
    *   `GZH_HEADERS`: 填入完整的请求头。
    *   `GZH_COOKIES`: 填入 Cookie 中的关键字段（如 `USER_TOKEN`, `yxfk_token`, `mobile` 等）。

## 🚀 运行方式

直接运行 Python 脚本：

```bash
python xjyd_tasks.py
```

脚本启动后会持续运行（挂机模式）：
*   寻宝/抽奖任务：每 15-20 分钟执行一次心跳保活。
*   权益中心任务：每天执行一次，自动休眠至次日早晨 8:00。
